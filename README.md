# CS2 External ESP v2.4.1

External wallhack and aim assist for Counter-Strike 2. Works in fullscreen and windowed mode.

## Features
- Glow ESP (enemy only)
- Aimbot with smooth control
- Triggerbot with delay settings
- Radar hack
- Bunnyhop assist

## Quick Start
1. Close CS2 completely
2. Run `setup.bat` as Administrator
3. Launch CS2 and join a match
4. Press F5 to toggle overlay

## Keybinds
| Key | Function |
|-----|----------|
| F5 | Toggle ESP |
| F6 | Toggle Aimbot |
| F7 | Toggle Triggerbot |
| END | Unload cheat |

## Offsets
Updated for CS2 build 1.40.4.1. If the cheat doesn't work after a game update, wait for new offsets.
